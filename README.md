# Ideal Systems Case
This project was created as a front-end assignment for Ideal Systems.
The basic premise was to develop a small web app that was focused on JavaScript & React.

Even though Css & styling were not required, some basic styling was added just to improve the ux of the result.

In the end, it consists of:
- React
- TypeScript
- Css

No other packages were used but some helper functions were created, as well as some TypeScript types. While both did add a little complexity, the overall result was kept pretty simple.

## Running the project
In order to run the project you need to take the following steps:
1. Run `pnpm install` in the terminal.  
    \**You can use a different package manager but then you will have to remove `pnpm-lock.yaml`.*

2. Copy the contents of `.env.example` to `.env`.  
    \**Make sure the value is set in `.env`.*

3. Run `pnpm start` in the terminal.  
    \**You can use a different package manager for this as well.*

4. Go to `localhost:3000` in your browser
